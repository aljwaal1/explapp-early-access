# ExplApp Early Access

موقع بسيط لعرض تطبيقات ExplApp المتاحة للتجربة قبل الإصدار الرسمي.

## الصفحات الحالية

- الصفحة الرئيسية: `index.html`
- تطبيق قائمة التسوق الذكية: `shopping-list.html`

## الروابط المستخدمة

- مجموعة المختبرين: https://groups.google.com/g/arabictesters
- اختبار تطبيق قائمة التسوق: https://play.google.com/apps/testing/com.explapp.shoppinglist

## تشغيل الموقع عبر GitHub Pages

من إعدادات المستودع:

Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: main → Folder: /root → Save
